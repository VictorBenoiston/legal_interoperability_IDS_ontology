    This ontology has as its purpose to provide a legal interoperability domain description in order to foster an unambiguous representation of policies, i.e., service contracts, within International Data Spaces architecture.

    As a part of a proposed legal interoperability framework, ans fostering FAIR principles, this ontology is available in an open-access GitHub repository (https://github.com/VictorBenoiston/legal_interoperability_IDS_ontology), which is composed of:

    - All available files from LegIOn-IDS
    -Reference ontology overview
    -Documentation
    -ISO and overall glossary of terms
    -Ontological Engineering Lifecycle
    -Results from SPARQL queries.

    Current version: v.1.1"
